# Smart Garden Platform (RU)

Русскоязычное веб-приложение для управления домашним умным садом/огородом и ведения дневника роста растений.

## Технологии
- PHP 8+
- MySQL 8+
- HTML5/CSS3/Vanilla JS (AJAX)
- PDO (prepared statements)

## Структура проекта
- `config/` конфигурация приложения и БД
- `public/` веб-корень (`index.php`, `assets`, `uploads`, `robots.txt`, `favicon.svg`)
- `app/Controllers` контроллеры модулей
- `app/Models` модели
- `app/Helpers` инфраструктура (Router, DB, Auth, CSRF, View)
- `app/Middleware` middleware
- `app/views` шаблоны страниц
- `routes/web.php` маршруты
- `sql/schema.sql` структура БД
- `sql/seed.sql` демо-данные
- `docs/coursework.md` Глава 1 и Глава 2 (академическая часть)

## Быстрый запуск (OpenServer/XAMPP)
1. Создайте БД и примените SQL:
   - `sql/schema.sql`
   - `sql/seed.sql`
2. Проверьте настройки БД в `config/config.php`.
3. Веб-корень сайта:
   - рекомендуемо: `public/`
   - альтернативно: корень проекта, тогда используется `index.php` прокси.
4. Откройте в браузере главную страницу.

## Демо-аккаунты
- Администратор: `admin@smartgarden.local` / `password`
- Пользователь: `user@smartgarden.local` / `password`
